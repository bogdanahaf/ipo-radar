Why this exists

Cheap alerts without a $29/month digest

Paid services bundle email, SMS, and “AI S-1” tiers. If your real need is simpler, a small pipeline on GitHub Actions plus a Telegram channel gets you most of the calendar signal for the cost of a free Alpha Vantage key and a bot token.

What IPO Radar covers

  • Next session preview — after the US close, a digest of filtered listings for the next market day (your “know who is going public tomorrow” beat).
  • Pre-open ping — on listing day, about an hour before the US cash session opens, a short reminder for the same filtered set.
  • Buzz meter — each row gets a simple buzzScore from name keywords, venue, price band, and your watchlist. It is narrative heat for sorting attention, not a model of first-day returns.
  • Watchlist bias — set IPO_WATCHLIST in GitHub variables to boost names you care about; everything still renders on this page for manual review.

What it does not cover

  • Broker-specific “stages” — lines like “allocation window open” or “IPO tonight after close” are workflow states inside Robinhood, SoFi, or other apps. They are not in the public IPO calendar feed this project uses; keep those notifications inside the broker if you still participate in allocations.
  • Schwab, IBKR, Robinhood hooks — no brokerage integration, no order flow, no advice. Use each firm’s own alerts, watchlists, and corporate-action notices alongside this calendar.

Educational only. Data comes from Alpha Vantage’s IPO calendar; verify filings and symbols with the exchange or SEC before you trade.
